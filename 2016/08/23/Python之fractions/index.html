Python之fractions

发表于 2016-08-23 | 分类于 python | 阅读次数:

Python 从2.6开始，增加了 fractions 模块，提供有理数（Rational numbers）的计算。

以下测试使用环境是 Python 2.7.11。

fractions 介绍

Python 2.6以前的版本的上线文档中的 Library Reference 里没有关于 fractions 模块的内容，所以，python 2.6以前的版本正常情况下无法使用 fractions 模块。

Python 2.6有关 fractions 的介绍

fractions 模块提供 Fraction 类来表示有理数，Fraction 有3种构造方式：由一对整数（一个分子，一个分母）构造；由另一个有理数对象构造；由字符串（如 ‘1/3’）构造。

第一种构造方法要求分子分母都是整数，且分母不为0，否则会产生一个 ZeroDivisionError 错误。

class fractions.Fraction(numerator=0, denominator=1)

第二种构造方法要求提供一个有理数（numbers.Rational 类型的）。

class fractions.Fraction(other_fraction)

第三种构造方法要求的字符串有两种格式：

[sign] numerator ['/' denominator] #sign是正负号('+'或'-')，numerator 和 denominator 是10进制整数
[sign] integer '.' [fraction] | [sign] '.' fraction

三种方法的示例：

>>> from fractions import Fraction
>>> Fraction(16, -10)
Fraction(-8, 5)

>>> Fraction(123) #分子为123，分母取默认值1
Fraction(123, 1)

>>> Fraction() #分子取默认值0，分母取默认值1
Fraction(0, 1)

>>> Fraction('3/7') #字符串构造法
Fraction(3, 7)

>>> Fraction(' -3/7 ') #字符串构造法，可以以空格开头或者结尾
Fraction(-3, 7)

>>> Fraction('1.414213 \t\n') #字符串构造法，使用一个10进制浮点数
Fraction(1414213, 1000000)

>>> Fraction('-.125') #字符串构造法
Fraction(-1, 8)

此外，在 Python 2.7.11（我使用的版本）中，fractions.Fraction 的类描述显示，该类对象还可以使用浮点数和10进制数构造。

Fractions can also be constructed from:
 |  
 |    - numeric strings similar to those accepted by the
 |      float constructor (for example, '-2.3' or '1e10')
 |  
 |    - strings of the form '123/456'
 |  
 |    - float and Decimal instances
 |  
 |    - other Rational instances (including integers)

fractions.Fraction 的使用

>>> a = fractions.Fraction('3/7')
>>> b = a + 1
>>> a
Fraction(3, 7)
>>> b
Fraction(10, 7)

>>> a = fractions.Fraction('3/7')
>>> b = a * 7
>>> a
Fraction(3, 7)
>>> b
Fraction(3, 1)
>>> c = int(b)
>>> c
3

特别的，当使用浮点数构造 fractions.Fraction 对象时，结果可能不准确，如下：

>>> c = fractions.Fraction(float(1)/float(3))
>>> c
Fraction(6004799503160661, 18014398509481984)

本意是构造 1/3，但结果不是，原因是构造参数1/3在计算机中的表示有误差导致最后的对象不是 Fraction(1, 3)，而变成了 Fraction(6004799503160661, 18014398509481984)，所以，正确的方法应该是：

>>> c = fractions.Fraction(1,3)
>>> c
Fraction(1, 3)

应避免使用浮点数构造 fractions.Fraction 对象。
