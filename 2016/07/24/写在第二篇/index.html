写在第二篇

发表于 2016-07-24 | 分类于 hexo | 阅读次数:

搞了快两周了，最后还是把github pages搭起来了，以后就可以用它来记东西咯，nice。

事实上，借助强大的工具（github + Hexo），不到一个星期就搭好了，并且可以在上面写东西了。至于剩下的那些时间，可以说是在探索（zhē téng），还真碰到很多问题，有些已经解决了，有些还没有。总而言之，略有收获，起码少虚度了几天光阴对吧。搭建的过程颇为曲折，后面有空再写出来。

2017/10/10:

终于有空把未完成的补充一下了。

windows环境

安装Node.js

下载Node.js，选择LTS(Long Term Support)版本。

安装Hexo

命令行（CMD)窗口下，执行：

npm install -g hexo

-g 参数表示全局安装，hexo 会安装到默认的目录中

新建博客目录并初始化

cd 到要创建存放博客的目录下，执行

hexo init myblog

会在该目录下创建一个名为 myblog 的文件夹，并初始化 hexo 到 myblog 文件夹内。

或者，在要创建存放博客的目录下新建一个 myblog 的文件夹，并 cd 到 myblog 文件夹内，执行

hexo init

也会初始化 hexo 到 myblog 文件夹内。

生成静态页面

cd 到新建好的 myblog 文件夹，执行

hexo g

以上命令完成了静态博客内容的生成，默认只有一篇 hello world 的文章，静态博客的内容在 myblog 文件夹内的 public 文件夹内。

预览效果

cd 到 myblog 文件夹内，执行 hexo s，有如下输出：

$ hexo s
INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

此时，可以在浏览器内，输入 http://localhost:4000/ 访问静态博客，访问 myblog 文件夹内的 public 文件夹内的博客内容。

如果执行 hexo s 报错误，或者访问博客失败，先确定下4000端口是否被禁用，或者被其他程序占用

部署到 github page

在这一步前，先确定电脑中安装了 git，没有的请先安装 git，git下载地址(windows版)

注册 github 账号

github网址，假设这里注册账号叫 myblog

新建静态博客仓库

新建一个 github 仓库，仓库名字为 账号名+.github.io，例如这里的 github 账号叫 myblog，则新建仓库名就是：myblog.github.io，仓库名不能是其他的格式，否则不能访问到。

添加SSH公钥到博客仓库的账户中

方法在这里

hexo To github page

打开 myblog 目录下的 _config.yml 文件，修改 deploy 内容，如下：

deploy:
      type: git
      repository: git@github.com:myblog/myblog.github.io.git

其中 type，表示使用提交使用 git 工具，repository 是远程仓库的地址，即新建的博客仓库地址，这个地址在仓库页面复制（Clone or download -> Use SSH -> Copy to clipboard）

cd 到 myblog 文件夹下，执行：

hexo d

显示如下：

$ hexo d
INFO  Deploying: git
INFO  Clearing .deploy_git folder...
INFO  Copying files from public folder...
INFO  Copying files from extend dirs...

。。。此处省略N行字

[master 4558336] Site updated: 2017-10-10 11:45:01
3 files changed, 42 insertions(+), 11 deletions(-)
warning: LF will be replaced by CRLF in page/4/index.html.
The file will have its original line endings in your working directory.
Branch master set up to track remote branch master from git@github.com:myblog/myblog.github.io.git.
To git@github.com:myblog/myblog.github.io.git
c53ef3f..4558336  HEAD -> master
INFO  Deploy done: git

成功部署到 github，输入网址就可以访问，https://myblog.github.io/

myblog是例子中使用的仓库名，访问静态博客的网址，根据创建静态博客仓库时使用的名字的不同而不同

如果使用 hexo d 命令失败，查看一下，电脑中是否安装了 git，没有的请先安装 git

至此 Hexo + github 搭建个人静态博客完成，可以写博客啦，新建博客放在 myblog -> source -> _posts 文件夹内，写好后，cd 到 myblog 文件内，执行 hexo g，生成新的博客内容，再执行 hexo d 部署到 github 就可以了。
