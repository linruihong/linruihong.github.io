Python之urllib

发表于 2017-09-14 | 分类于 python | 阅读次数:

官网的描述：

This module provides a high-level interface for fetching data across the World Wide Web.

翻译：

该模块提供了通过万维网获取数据的高级接口。

使用 urllib 或 urllib2 可以方便地使用多种网络协议，获取网络数据。

一般的使用方法网上有很多介绍，这里记一下学习中的小问题。

之前接第三方平台API时，按文档要求，是要把数据先编码后才能发送的。网上找了一遍后发现可以用 urllib 的 urlencode 和 quote（对应的解码是unquote）来做这个事情。

使用中发现，urlencode 和 quote 都会对%进行转码（转成%25），如果先使用 quote 转码str="%"，然后再用 urlencode 转码包含 str 的dict={1:str}，则会出现结果为1=%2525。

原因是用一个函数去转码另一个函数的返回值，导致“%”号被转码了2次。

这种结果有时是非必要的，即有时只需要转一次就够了，要注意，这种情况下，两个函数只能选用其中一个。
