整数红包随机算法，试了三版，前面的随机效果都不理想，最后一版根据网上方法改写。

第一版

#一次生成所有红包List(按金额总数、红包个数、最小红包金额、最大红包金额)
def RedPacketList(amount, count, minm, maxm, factor = 2):
    reminded = amount - count * minm
    delta = maxm - minm
    if reminded < 0 or delta < 0 or count <= 0 or reminded > delta * count:
        return []

    lst = []
    for idx in xrange(count - 1):
        gold = rand_int(max(0, reminded - delta * (count - 1 - idx)), min(int(1.0 * factor * reminded / (count - idx)), delta))
        lst.append(minm + gold)
        reminded -= gold

    lst.append(minm + reminded)
    return lst

第二版

#根据第一版改写，目的是一次输出一个红包，而不是一次生成所有红包List
def Redpacket(amount, number, minm, maxm):
    if number == 1:
        return amount
    reminded = amount - number * minm
    delta = maxm - minm
    gold = randint(max(0, reminded - delta * (number)), min(int(3.0  * reminded / (number+1)), delta))
    return minm + gold
