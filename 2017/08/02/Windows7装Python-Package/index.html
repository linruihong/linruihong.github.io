Windows7装Python Package

发表于 2017-08-02 | 分类于 python | 阅读次数:

Windows7下安装Package的方法。

安装setuptools

下载setuptools 36.2.0，解压后放在 Python 安装目录下，CMD 进入 setuptools 目录，运行：

python setup.py install

完成安装。

如果提示 python 不是内部命令，先把 Python 安装目录加入到环境变量中，再重新 CMD 进入 setuptools 目录，执行刚才的命令。

安装pip

下载pip 9.0.1，解压后放在 Python 安装目录下，CMD 进入 pip 目录，运行：

python setup.py install

安装完成。

如果提示 python 不是内部命令，先把 Python 安装目录加入到环境变量中，再重新 CMD 进入 setuptools 目录，执行刚才的命令。

安装完成后，把 Python 目录下的 Scripts 目录也加入到环境变量中，否则运行 pip 命令会提示“pip 不是内部命令”。

安装BeautifulSoup

下载beautifulsoup4 4.6.0，解压后放在 Python 安装目录下，CMD 进入 beautifulsoup 目录，运行：

setup.py build
setup.py install

安装完成。

在 CMD 下运行 Python，测试是否安装成功：

c:\Python26\beautifulsoup4-4.6.0>python
Python 2.6.6 (r266:84297, Aug 24 2010, 18:13:38) [MSC v.1500 64 bit (AMD64)] on
win32
Type "help", "copyright", "credits" or "license" for more information.
>>>
>>> from bs4 import BeautifulSoup
>>>

没有报错即安装成功
