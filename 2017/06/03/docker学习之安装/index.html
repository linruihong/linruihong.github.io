docker学习之安装

发表于 2017-06-03 | 分类于 docker | 阅读次数:

记一下 Windows 7 下安装过程的坑⊙﹏⊙‖∣

不看踩坑过程，可直接跳过，看总结。

docker 这东西，听了好多次，就是没去了解是什么，但是感觉好厉害的样子–!。趁着有空闲时间，搞一下:-)。

不管3721，先把环境搞起来。网上虽然有教程，但是要么太旧，要么太傻，我又喜欢搞自定义安装，结果就是安装个 docker 花了我1个小时。

抱着玩玩的心态，选择在 windows 上安装，按《Docker 技术入门与实战》（应该是第1版）上面说的，在 windows 上运行 docker 要额外使用一个虚拟机来提供 Linux 支持，推荐使用 Boot2Docker 工具。

然而，到书上提供的网址docker docs(docker 手册)指引下载了Docker for Windows后发现不能安装（提示：This version of Docker requires Windows 10 Pro , Enterprise or Education edition with a minimum build number of 10586 . Please use Docker Toolbox），只能到Docker官网下载Docker Toolbox。

下载完 Docker Toolbox 后，安装的时候，并没有按默认方式安装（软件自带的 git 我没有选择安装，因为机器上本来就装过 git 了，想发再装也是多余，安装路径也不是默认的而是指定我自己的路径），接着安装完后，第一次打开 Docker Quickstart Terminal 快捷方式的时候，提示：windows正在查找bash.exe。如果想亲自查找文件，请单击“浏览”。

想着可能是没有安装软件自带的 git ，于是又重新装了一次，还让软件把机器原来的 git 给卸了，再开，还是提示：windows正在查找bash.exe。如果想亲自查找文件，请单击“浏览”。网上找了一下，才知道是快捷方式中指定的执行程序 bash.exe 找不到了，只要找到 git 安装目录下的 bin 目录下的 bash.exe 后，再指定给快捷方式，就好了。设置完再开，没有提示找不到 bash.exe 了，进入 CMD 了，不过输出内容跟网上教程描述的不一样，是这样的：

然后就到网上搜，知道是本地没有默认的 Boot2Docker ISO 文件，需要联网下载，但是等了好久还是下载失败了。然后就把下载地址复制出来，用浏览器下，等了好久都下不了，想着可能是“局域网”不允许下载吧，挂VPN，还是不行。快放弃了的时候换了个浏览器（开始一直用 360，后来换谷歌 Chrome），居然下得动了。

下好后，把下好的 boot2docker.iso 文件，放到 CMD 里面提示的下载目录(即 C:\Users\Administrator\.docker\machine\cache\)中，再次打开 Docker Quickstart Terminal ，成功了。

注：在github上看到 boot2docker 貌似有个 windows 安装程序 windows-installer ，现在已经下载不了了，github上的 Lastest release 写着 Deprecated：This project (the boot2docker Windows Installer) is officially deprecated in favor of the new Docker Toolbox 。取而代之的是 Docker Toolbox（下载地址）

总结：
windows 7 下安装docker的最简单方式：查看Docker Toolbox，下载Docker Toolbox，按默认方式一路安装到底。

可能遇到的问题及解决办法：

  • 打开 Docker Quickstart Terminal 快捷方式，提示 windows正在查找bash.exe。如果想亲自查找文件，请单击“浏览” ，这是因为快捷方式里设置的 base.exe 路径不对，导致找不到 bash.exe 。右键 Docker Quickstart Terminal 快捷方式 -> 属性，在本机上找到正确的 bash.exe 路径，重新设置就好了。
  • CMD 下提示 Downloading XXXXXX /boot2docker.iso... ，这是因为本机没有 Boot2Docker ISO 镜像文件，正常情况下会自动下载完成的，等等就好。也可以把提示中的下载地址复制出来单独下载，然后把下载好的文件放入提示中出现的下载目录中，再启动 Docker Quickstart Terminal 快捷方式。具体做法是：复制下载地址（本例中是： https://github.com/boot2docker/boot2docker/releases ）在浏览器中打开，选择里面的 Lastest release 版本下载，然后把下载好的 boot2docker.iso 文件，放到指定的下载目录（本例中是： C:\Users\Administrator\.docker\machine\cache\），打开 Docker Quickstart Terminal 快捷方式。

windows 10 下也可以下载Docker Toolbox，按默认方式一路安装到底，与 windows 7 无异。

2017/6/10 补充：windows 10 下第一次安装完成，启动后，报 VirtualBox 启动失败。然后把 Docker Toolbox 和 VirtualBox 都卸载了，重新安装一次，居然好了。汗…

另外，windows 10 系统下，可以查看docker docs，下载Docker for Windows，不过貌似会要求开启 windows 10 的 Hyper-V 服务，Hyper-V 服务听说会和 VBOX、VMware 等冲突，且开启后再想关闭不一定可以，不敢试。不小心开了的想关掉可试试下面的方法（没试过，不保证成功）：
